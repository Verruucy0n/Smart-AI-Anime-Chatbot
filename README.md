# Smart AI Anime Chatbot
An AI Chatbot named Pomona-chan, a tsundere high school student that can explain and quiz Math and Science related topics. 
This project got me into the Top 10 in Sokrates SPARK 2026, a national event and competition organized by Sokrates Empowering School

## Programming Languages & Tools:
- Javascript, HTML, CSS
- The Gemini API
- Procreate (For the Sprites)

## How to use in VS Code
1. Install the "Live Server" Extension
2. Right click on index.html, then click "Open with Live Server"
